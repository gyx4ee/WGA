# WinSys Guardian Advanced

Начален Python prototype за дипломна работа, изграден изцяло с `tkinter` и готов за работа в `PyCharm`.

## Стартиране

```bash
python app.py
```

## Какво има в момента

- стартов екран, близък до показаната визия
- зелен loading bar в центъра
- плавна анимация на зареждането
- прогрес, който следва реални startup задачи, а не само фиксиран таймер

## Как работи loading bar-ът

Барът не проверява директно "колко е бърз компютърът". Вместо това се движи според реални стъпки при стартиране:

- зареждане на конфигурация
- събиране на системна информация
- проверка на файлове в проекта
- подготовка на вътрешни модули

Ако компютърът е по-бърз, тези стъпки приключват по-бързо и барът стига до 100% по-рано.
